# 项目分析：

# 一、<span class="font" style="font-family: 方正粗黑宋简体"> 结构部分 </span>

# 参考灯哥四足机器人结构制作，附件为最新版的 3D 图纸，可直接打样生产，生产价格为 200 元左右。 具体开源链接点击这里：灯哥开源四足机器人项目

# 二、电路部分

# 1、基于鸿蒙 Hi3861 主控，支持鸿蒙系统开发。采购链接

# 2、8 路舵机采用驱动芯片 PCA9685PW 来管理，此芯片可控制多达 16 路的舵机。采用 IIC 接口与单片机连接。

# 3、MPU6050 模块通过读取角速度，陀螺仪数据来识别狗狗当前的状态，从而更好的控制狗狗行走。

# 三、软件部分

# 1、采用鸿蒙开发平台进行开发，开发环境为 ubuntu20 Linux 环境编译，windows 平台串口烧录程序
